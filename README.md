# 主要功能
* 登录系统
* 主页查看课程数据
* 用户管理页增加、删除、编辑、查找用户
-------

# 技术栈
* js、html5、css3编写页面
* vue全家桶进行路由映射，状态管理，完成单页面富应用
* element UI快速布局
* echarts 绘制图表
* mockjs 模拟后端接口返回数据
